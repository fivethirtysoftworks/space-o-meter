Space-O-Meter

A lightweight macOS menu bar utility for monitoring disk space on local and mounted network volumes.

Space-O-Meter keeps an eye on your storage so you don’t run out of disk space unexpectedly. It lives in the macOS menu bar and provides a clean, focused view of available storage without turning your menu bar into a full system monitor.

⸻

Features

• Menu Bar Disk Monitoring
See available storage directly from the macOS menu bar.

• Local Disk Monitoring
Quickly check the status of your internal system drive.

• External Drive Support
Automatically detects and displays mounted external disks.

• Network Volume Monitoring
Displays usage for mounted network shares such as NAS or SMB volumes.

• Simple Storage Breakdown
View used, free, and total capacity at a glance.

• Lightweight Design
Built to do one job well without unnecessary system monitoring features.

⸻

Why Space-O-Meter Exists

Many system monitoring tools include disk usage as one feature among dozens. Space-O-Meter focuses specifically on storage awareness.

The goal is simple:

Know how much disk space you have left before it becomes a problem.

Instead of graphs, dashboards, and excessive metrics, Space-O-Meter provides a clear storage overview with minimal overhead.

⸻

System Requirements

• macOS 13 Ventura or later
• Apple Silicon or Intel Mac
• Mounted local, external, or network disks for monitoring

Network disk monitoring only works for volumes already mounted in macOS.

Usage

After launching Space-O-Meter:
	1.	The app appears in the macOS menu bar.
	2.	Click the icon to view storage usage.
	3.	The menu shows:
	•	Local disk usage
	•	Mounted network volumes
	•	External drives
	4.	Open Settings from the menu to adjust monitoring options.
